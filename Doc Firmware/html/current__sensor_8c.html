Detailed Description

Implements functionality for reading the current sensor.

Author
Nicolás Almaraz

Current Sensor Calculations: Sensitivity: S = 185mV/A |--> 1mV = 1.2409 counts |--> S = 229.5681 counts / A

counts = S * current |--> current = counts * 1/S |--> 1/S = 0.004356004 A / counts |--> 1/S = 4.356004 mA / counts
